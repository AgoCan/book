# 周末班与全日制班级的区别？
周末班主要面向在职人员，多为已经是IT相关从业人员为主。旨在通过业余时间提升，或额外学习掌握一项技能。  

全日制班为集中训练，有较强的目的性和结果导向。同时投入的精力，财力也最大。是对那些渴望转型，渴望学习掌握一项IT技能的小伙伴的最佳选择。
